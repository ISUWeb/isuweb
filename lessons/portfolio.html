# Portfolio

In this lesson you will create a static site and host it on GitHub. A text editor with autocomplete and syntax highlighting is recommended.

TIP

Sublime has a portable version that can be used in the lab computers.

# Creating the site

Create your site. Below is a basic example.





Â 









<!DOCTYPE html>
<html lang="en">
<head>
    <title>Portfolio</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="container">
        <h1>Portfolio</h1>
        <p>Welcome to my portfolio</p>
    </div>
</body>
</html>

# Hosting locally

You can open the HTML file in a web browser.

Or you can start a server that will host your site locally using Python:

python -m SimpleHTTPServer 8000

# Delpoying to GitHub

  1. Log in to GitHub and create a repository
  2. Follow the instructions for pushing an existing repository
  3. Enable GitHub Pages by following the steps in their documentation
  4. Visit your site
